Casualties: Unknown - Tarkov-Style Medical Mod

Version: 0.3.0

A BepInEx mod for Casualties: Unknown Demo that brings the complete Escape from Tarkov medical system into the game - 16 combat stimulant injectors, 12 medical items, and a fully integrated medical framework.

Each stimulant has its own unique buff/side-effect mechanics, registered into the game's native item table via reflection. Medical kits hook into native minigame systems (BandageMinigame, SyringeMinigame, ShrapnelMinigame, DislocationMinigame).

---

REQUIREMENTS

- Game: Casualties: Unknown Demo (Steam)
- Mod Loader: BepInEx 5.x

Installing BepInEx:
1. Download BepInEx 5.x stable (x64) from https://github.com/BepInEx/BepInEx/releases
2. Extract into your game root folder (Steam/steamapps/common/Casualties Unknown Demo/)
3. Launch the game once to generate the BepInEx/ folder, then close it

---

INSTALLATION

1. Download CUTarkovMedicalMod_v0.3.0.zip
2. Extract the contents into your game root folder
   - After extraction:
     [Game Root]/BepInEx/plugins/CUTarkovMedicalMod/
       - CUTarkovMedicalMod.dll
       - Framework/Assets/  (icons + sound effects)
       - Lang/              (EN.json, zh_CN.json)
3. Launch the game

Verify: check BepInEx/LogOutput.log for "Casualties: Unknown - Tarkov-Style Medical Mod loaded. Enabled=True"

FEATURES

- 16 custom stimulants with unique buffs and side effects
- 12 medical items (first aid kits, surgical kits, balms, pills)
- Starting loadout: 5 fixed medical items + random extras
- World loot and container drops (medical crates, supply crates, corpses)
- Buff indicator with countdown timers (MoodleManager)
- Tunnel vision post-processing for stim side effects
- Custom injector sound effect
- Hold SHIFT to expand item effect descriptions
- Console spawn command support for all custom items
- Multiplayer compatible (auto-detects KrokMP)
- Bilingual: English and Simplified Chinese

CONTROLS

- Left Click (inventory) - Use stimulant / drink medicine
- Left Click (body limb) - Use medical kit / surgical kit / balm
- Hold SHIFT - Expand item effect details
- F7 / Numpad 7 - Debug status output

---

CONSOLE SPAWN

All items spawnable via: spawn [itemkey]
Examples: spawn etg_c, spawn cu_morphine, spawn grizzlykit, spawn cms

---

CONFIGURATION

Auto-generated at BepInEx/config/com.yourname.cu.tarkovmedicalmod.cfg

- EnableMod (true) - Master toggle
- FeatureMode (Both) - Disabled / StartingLoadoutOnly / WorldLootOnly / Both
- CompatibilityMode (AutoSafe) - KrokMP detection strategy
- StartingLoadout MinItems/MaxItems (1/3) - Random item count range
- WorldLoot MinItems/MaxItems (1/4) - World loot count range
- AllowDuplicateItems (true) - Allow duplicate items
- Seed (0) - Random seed (0 = random)

---

COMPATIBILITY
